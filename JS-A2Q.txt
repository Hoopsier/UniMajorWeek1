1Q Why does JavaScript output undefined instead of throwing an error in the following code?

console.log(message);

var message = 'Hi there!';


1A you used var, which is deprecated due to it being a pain in the ass to debug for it being in scope earlier than when defined, leaving you with an undefined.



2Q Why does JavaScript throw an error instead of logging undefined in the following code?

console.log(message);

let message = 'Hi there!';

2A let is the new version of var, which isn't defined in scope until after the line it gets defined.
